# Screenshot Folder

This folder is reserved for the screenshot pack referenced by:

- [strehe-prona-user-admin-manual.md](../strehe-prona-user-admin-manual.md)

## Capture guidance

- Capture pages using a seeded demo environment.
- Prefer one clean desktop capture per page.
- Use stable data so screenshots do not become confusing a week later.
- Save files using the filenames listed in the manual appendix.

## Minimum roles to use during capture

- `admin`
- `office`
- `field`
- `contractor`

## Notes

Authentication and password screens should be captured with non-sensitive demo accounts only.
